# `.exp/` — 可复用经验

本目录存放从项目开发中提炼出的通用经验文档，供当前和未来项目复用。

## 建议结构

```text
.exp/
├── README.md
├── frontend/
├── backend/
├── testing/
└── operations/
```

每个主题建议成对维护：

| 前缀 | 读者 | 内容 |
|---|---|---|
| `Human_` | 人类开发者 | 背景、原理、决策思路和踩坑经验 |
| `Agent_` | AI 助手 | 可执行规则、反例、验证方式和复用模板 |

新增经验后，同步更新本文件或对应子目录索引。
