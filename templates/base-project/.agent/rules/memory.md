# 长期上下文规则

## `.ai_memory/`

`.ai_memory/` 用于存放项目维护者确认可记录的长期上下文。模板默认只提供说明和空占位，不包含真实内容。

## 建议文件

| 文件 | 写入模式 | 用途 |
|---|---|---|
| `1_project_context.md` | 只读为主 | 项目目标、架构决策和不变量 |
| `2_active_task.md` | 覆写 | 当前任务、进度、下一步和阻塞项 |
| `3_work_log.md` | 追加 | 日常开发流水账 |
| `0_archive_context.md` | 追加 | 阶段性归档和决策演变 |

## 使用原则

- 只有在项目维护者确认需要长期留痕时才创建具体记忆文件。
- 不记录真实凭据、个人信息、不可公开流程或本地机器细节。
- 归档时优先写稳定事实和可复用决策，避免记录临时噪声。
