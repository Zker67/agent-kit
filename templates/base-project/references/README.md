# 外部参考索引

> 本目录记录项目可参考的外部文档、标准、教程、代码库和实现案例。它不是当前项目事实源。

## 一层入口

- [外部文档](./external-docs/README.md)
- [外部仓库](./external-repos/README.md)

## 记录要求

每条重要参考至少说明：

- 来源链接。
- 访问日期、版本号、commit 或 release。
- 许可证或使用边界。
- 与当前项目的关系。
- 已采纳、未采纳或待评估的结论。

## 使用规则

- 只登记必要摘要和结构，不默认复制外部全文。
- 外部资料影响当前实现后，应把最终项目事实写回 `docs/` 或源码。
- 外部仓库可以记录目录结构和借鉴点，但不要把第三方仓库、lockfile、构建产物或管理器状态直接放进本目录。
