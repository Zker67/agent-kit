# 外部文档

> 本目录记录外部官方文档、标准、教程、论文或资料站点。

## 建议结构

重要资料可以按来源建子目录：

```text
references/external-docs/
├── README.md
└── source-slug/
    ├── README.md        # 来源、版本、许可、适用范围
    ├── structure.md     # 上游文档结构或目录摘要
    ├── notes.md         # 与当前项目相关的摘录和判断
    └── snapshots/       # 可选，小段合规摘录或本地生成摘要
```

只是一条轻量链接时，可以先登记在下面的表格；值得长期跟踪时再建子目录。
